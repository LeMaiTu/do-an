## Giới Thiệu

## Build src
- `composer install`
- `composer update`
- `cp .env.example .env`
- `php artisan key:generate`
- `php artisan migrate:fresh --seed`
- `php artisan serve`

## Run npm
- `npm install`
- `npm run watch`
